.Passos
. Certifique-se de que não haja operações de E/S ocorrendo entre o sistema de storage e todos os hosts conectados. Por exemplo, você pode executar estas etapas:
+
** Parar todos os processos que envolvem os LUNs mapeados do armazenamento para os hosts.
** Garantir que nenhuma aplicação esteja gravando dados em LUNs mapeados do storage para os hosts.
** Desmonte todos os sistemas de arquivos associados aos LUNs no sistema de storage.
+

NOTE: Os passos exatos para interromper as operações de E/S do host dependem do sistema operacional do host e da configuração, o que está além do escopo destas instruções. Se você não tiver certeza de como interromper as operações de E/S do host em seu ambiente, considere desligar o host ou entre em contato https://mysupport.netapp.com/site/global/dashboard["Suporte à NetApp"^].

+

CAUTION: *Possível perda de dados* -- se você continuar este procedimento enquanto as operações de e/S estão ocorrendo, você pode perder dados.



. Em cada controlador, certifique-se de que o LED NV Caching Active (verde) esteja apagado.
+
Quando o LED NV Caching Active (verde) estiver apagado, qualquer dado na memória cache foi gravado nas unidades e é seguro prosseguir com este procedimento.

+

NOTE: Se o LED NV Caching Active (verde) estiver aceso, os dados em cache estão sendo gravados nas unidades. Você deve aguardar a conclusão do processo e o LED NV Caching Active (verde) se apagar. No entanto, se o LED permanecer aceso por mais de cinco minutos, entre em contato  https://mysupport.netapp.com/site/global/dashboard["Suporte à NetApp"] antes de prosseguir com este procedimento.

+
O LED NV Caching Active (verde) está localizado próximo ao ícone NV no controlador.

+
image::../media/drw_g_nvmem_led_ieops-1839.svg[Localização do LED de status NV]

+

. Confirme se as operações nos controladores foram concluídas, usando SANtricity System Manager:
+
.. Selecione *Storage* > *Operations in progress* para ver a tela Overview.
.. Aguarde até que todas as operações sejam concluídas e a tela Overview exiba *No operations currently in progress*, antes de prosseguir para a próxima etapa.


. Exclua todos os hosts definidos do sistema de storage.
+
Para obter instruções, consulte o https://docs.netapp.com/us-en/e-series-santricity/sm-storage/delete-host-or-host-cluster.html["Excluir um host ou cluster de hosts no SANtricity System Manager"^] procedimento.
