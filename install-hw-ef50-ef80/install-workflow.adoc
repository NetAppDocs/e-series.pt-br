= Instalação e configuração - EF50 e EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para instalar e configurar seu sistema de storage EF50 ou EF80, você deve preparar o local, instalar e conectar os componentes de hardware, ligar o sistema de storage e concluir a configuração do sistema de storage.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Um"]link:install-prepare.html["Prepare-se para a instalação"]
[role="quick-margin-para"]
Para se preparar para instalação, organize o local, entenda as precauções de içamento do seu sistema de storage, desembale as caixas e compare o conteúdo com o packing slip, e registre o sistema de storage para acessar os benefícios de suporte.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dois"]link:install-hardware.html["Instale o hardware"]
[role="quick-margin-para"]
Para instalar o hardware, instale o hardware de montagem em rack no gabinete ou rack e, em seguida, instale e fixe seu sistema de storage.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Três"]link:install-cable.html["Conecte o hardware"]
[role="quick-margin-para"]
Para conectar o hardware, conecte os controladores entre si para habilitar o espelhamento entre controladores, o que garante redundância total do sistema e é usado para espelhamento de cache e envio de E/S, e então conecte os controladores à sua rede host.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quatro"]link:install-power-hardware.html["Ligue o sistema de storage"]
[role="quick-margin-para"]
Para ligar seu sistema de storage, conecte um cabo de força em cada fonte de alimentação do controlador e, em seguida, conecte-os a fontes de energia em circuitos diferentes

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinco"]link:install-complete-setup.html["Conclua a configuração do sistema de storage"]
[role="quick-margin-para"]
Para concluir a configuração do seu sistema de storage, conecte e configure as portas de gerenciamento nos controladores, configure e gerencie seu sistema de storage e, em seguida, verifique se o protocolo do módulo de E/S do host está correto.
